### Bug 2: Video/Audio Playback Stops Prematurely ✅

- Fixed error handling for `file.Seek()` operation (was silently ignored)
- Added error handling for `io.Copy()` and `io.CopyN()` operations

### Bug 3: Persistent Image Rotation ✅
- Integrated `github.com/dsoprea/go-jpeg-image-structure/v2` library for EXIF handling
- Modified `internal/imaging/rotate.go` to strip EXIF data after JPEG rotation
- The `stripExifAndWrite` function uses `SegmentList.DropExif()` to remove EXIF segments
- This prevents browsers from applying EXIF orientation on top of pixel rotation
- Added test `TestRotateImage_StripsExifOrientation` to verify the fix
